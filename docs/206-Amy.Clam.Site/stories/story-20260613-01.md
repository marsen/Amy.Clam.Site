# 客戶 → 在 LINE LIFF 瀏覽商品列表與今日天氣

**日期**：2026-06-13
**想法**：#206

## 互動關係

```
客戶開啟 LINE LIFF
  → 顯示今日天氣（氣溫、天氣狀況）
  → 顯示商品列表（名稱、價格、庫存狀態：有貨 / 售完）
```

**邊界條件**
- 已下架商品不顯示
- 天氣 API 失敗時，天氣區塊顯示「天氣資訊暫時無法取得」，不影響購物
