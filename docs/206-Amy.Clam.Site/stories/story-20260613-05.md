# 管理者 → 管理商品（價格與上下架）

**日期**：2026-06-13
**想法**：#206

## 互動關係

```
管理者進入商品管理頁
  → 看到商品列表（名稱、現有價格、狀態）
  → 點「編輯」
      - 修改價格 → 儲存 → 立即生效（前台即時反映）
  → 切換上架 / 下架開關
      - 下架後前台立即隱藏
      - 上架後前台立即顯示
```

**邊界條件**
- 價格須為正整數，不得為空或 0
- 下架不刪除商品，只隱藏
