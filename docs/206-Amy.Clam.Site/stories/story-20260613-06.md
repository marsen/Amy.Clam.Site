# 管理者 → 查看銷售報表

**日期**：2026-06-13
**想法**：#206

## 互動關係

```
管理者進入銷售報表頁
  → 選擇日期區間（起始日 ~ 結束日）
  → 顯示：
      - 總營收（金額）
      - 各商品銷售數量 + 小計金額（表格）
      - 每日營收折線圖（X 軸：日期，Y 軸：金額）
```

**邊界條件**
- 區間內無訂單時顯示空白報表（總營收 $0）
- 預設顯示本週資料
