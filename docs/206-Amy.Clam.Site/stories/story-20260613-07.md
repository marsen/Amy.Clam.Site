# 管理者 → 查看訂單明細列表

**日期**：2026-06-13
**想法**：#206

## 互動關係

```
管理者進入訂單列表頁
  → 顯示所有訂單（下單時間、客戶姓名、電話、取貨時間、金額、狀態）
  → 可依日期或狀態篩選
  → 點入單筆訂單
      - 顯示完整詳情（含商品明細、下單時天氣）
      - 可手動更新訂單狀態（待取貨 → 已取貨 / 已取消）
```

**邊界條件**
- 訂單狀態：待取貨 / 已取貨 / 已取消
- 已取消訂單不計入銷售報表
