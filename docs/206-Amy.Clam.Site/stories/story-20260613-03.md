# 客戶 → 用電話查詢歷史訂單

**日期**：2026-06-13
**想法**：#206

## 互動關係

```
客戶進入「查詢訂單」頁
  → 輸入電話號碼
  → 顯示該電話的所有歷史訂單列表
      - 訂單編號、日期、取貨時間、狀態
  → 點入單筆可看詳情
```

**邊界條件**
- 查無資料時顯示「找不到此電話的訂單記錄」
- 不需要密碼，電話即為查詢憑證
