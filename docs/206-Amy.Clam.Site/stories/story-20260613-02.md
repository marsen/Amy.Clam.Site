# 客戶 → 下單並填寫自取資訊

**日期**：2026-06-13
**想法**：#206

## 互動關係

```
客戶選擇商品與數量
  → 點「下單」
  → 選擇取貨地點
      ○ 新店（早上 08:00~12:00）
      ○ 內湖（下午 15:30~18:30）
  → 依地點顯示可選時段
      新店：08:00 / 09:00 / 10:00 / 11:00 / 12:00
      內湖：15:30 / 16:00 / 16:30 / 17:00 / 17:30 / 18:00 / 18:30
  → 填寫取貨資訊
      - 姓名（必填）
      - 電話（必填，格式驗證）
      - 取貨日期（必填）
      - 取貨時間（從上方時段選擇，必填）
  → 確認送出
  → 系統建立訂單，自動記錄該地點當時天氣
  → 顯示「下單成功，訂單編號 #XXXX」
  → LINE 聊天室自動送出訂單確認訊息（含訂單編號、地點、時間、金額）
```

**邊界條件**
- 電話未填或格式錯誤 → 不得送出，顯示錯誤訊息
- 售完商品不可加入購物車
- 同一客戶可重複下單
