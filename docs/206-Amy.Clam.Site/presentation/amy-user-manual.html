Chapter 1

客戶如何開始使用

LINE 官方帳號
帳號名稱Amy 蛤蜊選物
LINE ID@908srkuq
加入連結https://line.me/R/ti/p/@908srkuq
1
掃 QR code 加 Amy 為 LINE 好友
打開 LINE → 掃一掃 → 掃描 Amy 蛤蜊選物的 QR code，點「加入好友」。
2
點聊天室底部的「立即訂購」按鈕
加入好友後，聊天室底部會出現選單。點「立即訂購」，系統會在 LINE 內開啟購物頁面。
3
選商品、填取貨資料、送出訂單
頁面內可瀏覽商品、查看今日天氣、選擇取貨地點與時段，填寫姓名與手機後送出。
4
收到訂單確認訊息
下單成功後，LINE 聊天室會自動出現訂單確認訊息，包含取貨地點、時間與金額。
💡 也可以直接用瀏覽器開啟 amy-clam-site.vercel.app，不需要透過 LINE，但訂單確認通知只會在 LINE 內顯示。

Chapter 3

查詢歷史訂單

客戶可用手機號碼查詢所有歷史訂單：

1
開啟訂單查詢頁面
購物頁面下方有「查詢訂單」連結，或直接開啟：
amy-clam-site.vercel.app/orders
2
輸入手機號碼查詢
填入下單時使用的手機號碼（09xxxxxxxx），即可看到所有歷史訂單與狀態。
Chapter 4

管理後台操作（Amy 專用）

後台登入資訊
網址amy-clam-site.vercel.app/admin
帳號【正式上線前請更換】

🔒 帳號密碼請勿外洩。後台可存取所有客戶資料與訂單，請妥善保管。
A
商品管理
點左上「商品管理」。可點擊價格欄位直接修改售價，或點狀態按鈕切換上架 / 下架。變更即時生效，客戶重新整理頁面即可看到更新。
amy-clam-site.vercel.app/admin/products
B
訂單列表
查看所有訂單，可依「待取貨 / 已取貨 / 已取消」篩選。右側下拉選單可直接更新訂單狀態。
amy-clam-site.vercel.app/admin/orders
C
銷售報表
選擇日期區間，查看每日營收折線圖與各商品銷售數量彙總。預設顯示近 30 天。
amy-clam-site.vercel.app/admin/reports
